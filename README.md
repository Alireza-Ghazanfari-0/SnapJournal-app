# 🚀 Project Overview | معرفی پروژه

This is a full-stack practice web application built to simulate a real-world social/content-sharing platform.
این پروژه یک وب‌اپلیکیشن تمرینی فول‌استک است که شبیه یک پلتفرم اشتراک‌گذاری محتوا طراحی شده است.

It focuses on authentication, database integration, and modern UI development.
تمرکز اصلی پروژه روی احراز هویت، اتصال به دیتابیس و توسعه رابط کاربری مدرن بوده است.

---

# 🧰 Tech Stack | تکنولوژی‌ها

* JavaScript
* HTML & CSS
* Tailwind CSS
* React
* Next.js (Pages Router)
* MongoDB Atlas
* NextAuth (JWT & Credentials Authentication)

---

# 🎯 Project Goals | اهداف پروژه

The main objective of this project was to gain practical experience with full-stack development concepts.
هدف این پروژه کسب تجربه عملی در توسعه فول‌استک بوده است.

Key learning areas include:

* Next.js full-stack architecture
* MongoDB database design and connection
* Authentication & authorization (JWT & Credentials)
* Session management with NextAuth
* Responsive UI development using Tailwind CSS

---

# ✨ Features | امکانات

* User registration and authentication system
* Secure login with JWT-based authentication
* Profile creation and profile management
* Password update functionality
* Create and upload posts (image + text support)
* View and explore posts from other users
* Private commenting system
* Fully responsive design (mobile, tablet, desktop)

---

# 📱 UI / UX

The interface is designed with simplicity and usability in mind.
رابط کاربری ساده، مدرن و کاربرپسند طراحی شده است.

The application is fully responsive and optimized for all screen sizes.
اپلیکیشن کاملاً ریسپانسیو و مناسب تمام دستگاه‌ها است.
