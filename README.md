# SnapJournal-app
این سایت یک پروژه تمرینی است که با JavaScript، HTML، CSS (Tailwind)، React، Next.js (Page Router)، MongoDB Atlas و NextAuth (JWT/Credentials) ساخته شده است. کاربر می‌تواند ثبت‌نام کند، پروفایل بسازد، پست منتشر کند، پست‌ها را ببیند و نظر خصوصی ارسال کند. کاملاً ریسپانسیو و نزدیک به یک اپلیکیشن حرفه‌ای است.
